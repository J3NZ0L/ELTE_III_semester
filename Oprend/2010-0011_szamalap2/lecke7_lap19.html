Szignálok

Példák a szignálok használatára egy C programozási nyelven írt programban

Feladat

Feladat: készítsünk egy egyszerű programot, amelyben figyelmen kívül hagyunk az első jelzés után minden jelzést!

Legyen a feladatunk az, hogy készítsünk C++ programozási nyelven írt programot Sigignore.c néven, amelyben a gyermekfolyamat küldjön a szülőjének gyors egymásutánban két SIGTERM szignált! A szülőfolyamat blokkolja a SIGTERM szignált, hajtson végre néhány kiírást, majd oldja fel a blokkolást! A blokkolás feloldása után a jelzéshez tartozó kezelőfüggvény majd azonnal végrehajtódik. Csak az első jelzést kezeljük le, az utána kapottat már hagyjuk figyelmen kívül.

//-------------------- // Sigignore.c //-------------------- #include <stdlib.h> #include <stdio.h> #include <unistd.h> // Például a fork függvény. #include <sys/types.h> // Például a pid_t típus. #include <signal.h> // Például a signal függvény. #include <sys/wait.h> // Például a wait, és a waitpid függvény. void Kezelo(int sig) { // A jelzéskezelő függvény fejsora kötött. signal(SIGTERM,SIG_IGN); // Ha jön egy következő SIGTERM, figyelmen kívül hagyjuk. printf("%i -s szamu jelzest kaptam \n", sig); sleep(10); // A jelzéskezelő függvény végrehajtása hosszú, tíz másodperc. printf("Jelzes vege\n"); fflush(stdout); } int main(int argc, char * argv[]) { int status; int i; signal(SIGTERM, Kezelo); // Beállítja a SIGTERM szignál kezelőjének a Kezelo függvényt. pid_t child = fork(); if (child > 0) { for ( i = 1; i <= 5; i++) { // Eközben érkezik majd a jelzés. sleep(1); printf("Szulo vagyok- futok\n"); fflush(stdout); } waitpid(child, &status, 0); } else { sleep(2); // Kis várakozás után küld a szülőnek egy jelzést. kill(getppid(),SIGTERM); // Egy SIGTERM jelzést küld. sleep(1); kill(getppid(),SIGTERM); // Újra jelzést küld – az előző biztosan nem fejeződött be. } return 0; } //--------------------

Megfigyelés

Látszik, hogy csak az első SIGTERM jelzést dolgozza fel, a másodikra nem reagál. A jelzéshez tartozó alapértelmezett kezelési módot az előre definiált SIG_DFL függvénnyel tehetjük meg. Ne felejtsük el, hogy a SIGTERM szignál eredetileg a folyamat terminálását eredményezi.
