Könyvtárfolyam pozíciójának beállítása újra a folyam elejére, a rewinddir függvény

#include <sys/types.h> #include <dirent.h> void rewinddir(DIR * konyvtarleiro);

A rewinddir függvény segítségével visszaállíthatjuk az elejére a könyvtárfolyamot (directory stream). Így megint az első könyvtárbejegyzést olvassa be a readdir függvény, ha újra lefuttatjuk a rewinddir függvény után. Nem állítja be az errno hibaértéket, és hibás művelet végrehajtás esetén nincs rögzítve a hatása a függvénynek, mivel nincs visszatérési értéke.

Példa a rewinddir függvény használatára:

DIR * d; struct dirent * de; d = opendir("konyvtarnev"); de = readdir(d); rewinddir(d); // Első körben deklarálunk egy „DIR*” típusú „d” nevű változót, ami // a könyvtárfolyamot (directory stream) ábrázolja, illetve deklarálunk // egy „dirent*” típusú „de” nevű változót. // Az opendir függvénnyel megnyitunk egy könyvtárfolyamot (directory // stream) a paraméterként átadott „konyvtarnev” nevű könyvtárból, // és az opendir függvényünk visszatérési értékként az előre // definiált (DIR*) típusú mutatót adja vissza a könyvtárbejegyzésekre. // Ezzel definiáljuk a „d” nevű változónkat. // A readdir függvény egy már megnyitott „d” könyvtárfolyamból // (directory stream) olvassa ki az első könyvtárbejegyzést és a readdir // függvényünk visszatérési értékként az előre definiált (dirent*) // típusú mutatót adja vissza a könyvtárbejegyzésre. // Ezzel definiáljuk a „de” nevű változónkat. // A rewinddir függvény segítségével visszaállíthatjuk az elejére // a könyvtárfolyamot (directory stream).

Könyvtárfolyam pozíciójának beállítása egy megadott pozícióra, a seekdir függvény

#include <sys/types.h> #include <dirent.h> void seekdir(DIR * konyvtarleiro, long pozicio);

A seekdir függvény végrehajtása azt eredményezi, hogy az első paraméterben megadott könyvtárfolyam (directory stream) pozícióját beállítja a második paraméterben megadott pozícióra, így a megadott pozíciótól olvassa be a könyvtárbejegyzést a readdir függvény, ha újra lefuttatjuk a rewinddir függvény után. A pozíció értéke egy korábbi telldir függvényhívással állapítható meg. Nem állítja be az errno hibaértéket, és hibás művelet végrehajtás esetén nincs rögzítve a hatása a függvénynek, mivel nincs visszatérési értéke.

Példa a seekdir használatára:

long int poz; DIR * d; struct dirent * de; d = opendir("konyvtarnev"); poz = telldir(d); seekdir(d, poz); de = readdir(d); // Első körben deklarálunk egy „DIR*” típusú „d” nevű változót, ami // a könyvtárfolyamot (directory stream) ábrázolja, és deklarálunk // egy „dirent*” típusú „de” nevű változót, illetve deklarálunk egy // „long int” típusú „poz” nevű változót. // Az opendir függvénnyel megnyitunk egy könyvtárfolyamot (directory // stream) a paraméterként átadott „konyvtarnev” nevű könyvtárból, // és az opendir függvényünk visszatérési értékként az előre // definiált (DIR*) típusú mutatót adja vissza a könyvtár bejegyzésekre. // Ezzel definiáljuk a „d” nevű változónkat. // A telldir függvény a paraméterében megadott "d” könyvtárfolyam // (directory stream) aktuális pozícióját adja vissza és ezt // követően a következő pozícióra áll. // A seekdir függvény végrehajtásakor a "d” könyvtárfolyam (directory // stream) pozícióját beállítja a második paraméterben megadott // „poz” pozícióra. // A readdir függvény egy már megnyitott „d” könyvtárfolyamból // (directory stream) olvassa ki az első könyvtárbejegyzést, és a readdir // függvényünk visszatérési értékként az előre definiált (dirent*) // típusú mutatót adja vissza a könyvtárbejegyzésre. // Ezzel definiáljuk a „de” nevű változónkat.

A könyvtárfolyam jelenlegi pozíciójának lekérdezése, a telldir függvény

#include <dirent.h> long int telldir(DIR * konyvtarleiro);

A telldir függvény a paraméterében megadott könyvtárfolyam (directory stream) aktuális pozícióját adja vissza. Nem állítja be az errno hibaértéket, és hibás művelet végrehajtás esetén nincs rögzítve a hatása a függvénynek, mivel visszatérési értéke csak sikeres végrehajtást követően van.

Példa a telldir használatára:

long int poz; DIR * d; struct dirent * de; d = opendir("konyvtarnev"); poz = telldir(d); de = readdir(d); // Első körben deklarálunk egy „DIR*” típusú „d” nevű változót, ami // a könyvtárfolyamot (directory stream) ábrázolja, és deklarálunk // egy „dirent*” típusú „de” nevű változót, illetve deklarálunk egy // „long int” típusú „poz” nevű változót. // Az opendir függvénnyel megnyitunk egy könyvtárfolyamot (directory // stream) a paraméterként átadott "konyvtarnev” nevű könyvtárból, // és az opendir függvényünk visszatérési értékként az előre // definiált (DIR*) típusú mutatót adja vissza a könyvtárbejegyzésekre. // Ezzel definiáljuk a „d” nevű változónkat. // A telldir függvény a paraméterében megadott „d” könyvtárfolyam // (directory stream) aktuális pozícióját adja vissza és ezt // követően a következő pozícióra áll. // A readdir függvény egy már megnyitott „d” könyvtárfolyamból // (directory stream) olvassa ki az első könyvtárbejegyzést, és a readdir // függvényünk visszatérési értékként az előre definiált (dirent*) // típusú mutatót adja vissza a könyvtárbejegyzésre. // Ezzel definiáljuk a „de” nevű változónkat.
