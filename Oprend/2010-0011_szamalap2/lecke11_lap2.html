Példa a megosztott memória használatára

Most nézzünk egy egyszerű példát a megosztott memóriahasználatról! A példaprogramban a folyamat létrehoz egy megosztott memóriát, csatlakozik hozzá, majd a fork hívással létrehoz egy gyerekfolyamatot.

A szülőfolyamat beleír egy szöveget a megosztott memóriába, a gyerekfolyamat pedig kiolvassa azt. A kiolvasás előtt a gyerekfolyamat vár egy másodpercet, hogy a szülőfolyamat azt be tudja írni. A példában semmit nem teszünk a versenyhelyzet elkerülésére, csak abban bízunk, hogy egy másodperc alatt már beírta az adatokat a szülőfolyamat.

#include <sys/ipc.h> #include <sys/shm.h> #include <sys/types.h> #include <sys/stat.h> #include <stdio.h> #include <stdlib.h> #include <string.h> #include <unistd.h> #include <wait.h> #define MEMSIZE 1024 int main (int argc, char * argv[]) { pid_t child; key_t kulcs; int sh_mem_id; char * s; /* A megosztott memória létrehozáskor adni kell egy kulcsot, amelyet az ftok függvénnyel generálhatunk. */ kulcs = ftok(argv[0], 1); /* A MEMSIZE méretű megosztott memória létrehozása, olvasási és írási jogosultsággal a felhasználó részére. */ h_mem_id = shmget(kulcs, MEMSIZE, IPC_CREAT | S_IRUSR | S_IWUSR); /* Csatlakozás a megosztott memóriához. */ s = shmat(sh_mem_id, NULL, 0); child = fork(); //Leválás a megosztott memóriáról if(child > 0){ char buffer[] = "Hello!\n"; strcpy(s, buffer); // Megvárja míg véget ér a gyerek folyamat majd törli a megosztott memóriát. shmdt(s); wait(NULL); shmctl(sh_mem_id, IPC_RMID, NULL); } else if ( child == 0 ) { sleep(1); printf("gyerek: %s", s); shmdt(s); } return 0; }
