A szülő- és a gyermekfolyamatok

Példák a szülő- és a gyermekfolyamatokra egy C programozási nyelven írt programban

Feladat

Feladat: hozzunk létre három gyermekfolyamatot, amelyek kiírják az azonosító számukat, és a szülőfolyamat megvárja mindegyik gyermekfolyamatának a befejeződését!

Legyen a feladatunk az, hogy készítsünk C programozási nyelven írt programot, amelyikben három gyermekfolyamat van, mindegyik írja ki a PID számát – ahogy az előző feladatban is! Módosítsuk úgy a kódot, hogy minden gyermekfolyamat befejeződését várjuk is meg! (Használjuk a waitpid függvényt, amelyben megadhatjuk azt is, hogy milyen PID számú gyermekre várunk.)

//-------------------- // Forkok2.c //-------------------- #include <stdlib.h> #include <stdio.h> #include <unistd.h> // Például a fork. #include <sys/types.h> // Például a pid_t típus. #include <sys/wait.h> // Például a wait, és a waitpid. int main() { int status; printf("A szulo %i PID szamu\n", getpid()); pid_t child1, child2; child1 = fork(); if (child1==0) { // Gyermekfolyamat. child2 = fork(); if (child2 == 0){ // Az első gyermek első gyermeke. printf("1.gyermek 1.gyermeke,-PID %i\n", getpid() ); } else { printf("1.gyermek szulo aga,-PID %i\n", getpid()); waitpid(child2, &status, 0); // Várunk az első gyermek első gyermekére. printf(„Az elso gyerek bevarta a gyereket\n ”); } } else { // A szülőfolyamat. printf("A szulo itt kezdodik,-PID %i\n", getpid()); child2 = fork(); // A gyermekfolyamat létrehozása. if (child2 == 0) { // A szülő második gyermeke. printf("A szulo 2.gyermeke, -PID %i\n", getpid() ); } else { waitpid(child2, &status, 0); // Várunk a második gyermekre. printf("A szulo (2.gyereket bevarta)-PID %i\n", getpid()); waitpid(child1, &status, 0); // Várunk a szülő első gyermekére. printf("A szulo (1.gyereket bevarta)-PID %i\n", getpid()); } } return 0; } //--------------------

A futtatás eredménye

>gcc Forkok2.c –o Forkok2 >./Forkok2 A szulo 12717 PID szamu 1.gyermek szulo aga, -PID 12718 1.gyermek 1.gyermeke, -PID 12719 Az első gyerek bevarta a gyereket A szulo itt kezdodik,-PID 12717 A szulo 2.gyermeke,- PID 12720 A szulo (2.gyermeket bevarta),-PID 12717 a szulo (1.gyermeket bevarta) –PID 12717

Megfigyelés:

A waitpid függvénynek a hatására, az egyes gyermekfolyamatokat a szülőfolyamatok bevárják. Az utolsó, minden futtatásnál, a szülőfolyamat első gyermekének a bevárása.

A folyamatok működése közben hozzá tudunk férni néhány fontos információhoz. Például lekérdezhetjük az adott folyamat PID számát, a gyermekfolyamatban a szülőfolyamat folyamatazonosítóját a getppid függvénnyel, a felhasználó azonosítóját a getuid, a csoportazonosítót pedig a getgid függvény meghívásának segítségével.
