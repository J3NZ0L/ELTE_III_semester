Feladat

Feladat: készítsünk egy egyszerű programot, amelyben a szignálkezelő függvényünket lassítjuk!

Legyen a feladatunk az, hogy készítsünk C++ programozási nyelven írt programot SigBlokk_1.c néven, amelyben a gyermekfolyamat küldjön a szülőjének gyors egymásutánban két SIGTERM szignált! A kezelőfüggvény működését lassítsuk sleep hívással!

//-------------------- // SigBlokk_1.c //-------------------- #include <stdlib.h> #include <stdio.h> #include <unistd.h> // Például a fork függvény. #include <sys/types.h> // Például a pid_t típus. #include <signal.h> // Például a signal függvény. #include <sys/wait.h> // Például a wait és a waitpid függvény. Void Kezelo(int sig) { // A jelzéskezelő függvény fejsora kötött. // Jelzéskezelő a SIGTERM szignálhoz. Printf(„%i –s szamu jelzest kaptam \n”, sig); sleep(10); // A jelzéskezelő függvény végrehajtása hosszú, tíz másodperc. Printf(„Jelzes vege\n”); fflush(stdout); } int main(int argc, char * argv[]) { int status; int i; signal(SIGTERM, Kezelo); // Beállítja a SIGTERM szignál kezelőjének a Kezelo függvényt. Pid_t child = fork(); if (child > 0) { for ( i = 1; i <= 5; i++) { // Közben érkezik majd a jelzés. Sleep(1); printf(„Szulo vagyok- mukodok\n”); fflush(stdout); } waitpid(child, &status, 0); } else { sleep(2); // Kis várakozás után küld a szülőnek egy jelzést. Kill(getppid(), SIGTERM); // A SIGTERM jelzést küldi. Sleep(1); kill(getppid(), SIGTERM); // Újra jelzést küld – az előző biztos nem fejeződött be. } return 0; } //--------------------

Megfigyelés

Mivel a második jelzés az első jelzés jelzéskezelőjének végrehajtása közben érkezett meg, ezért várakozni kényszerül. Amint az első jelzéskezelő befejeződött, azonnal az addig blokkolt jelzés jelzéskezelője fut le, és csak ezután folytatódik a program.

Jelzést blokkolni mi magunk is tudunk. Ehhez azonban meg kell ismerkednünk a gyakorlatban is a jelzéscsoportokkal. Jelzéscsoportot létrehozni a sigemptyset, illetve a sigfillset függvénnyel lehet, amelyekkel egy üres, illetve egy minden jelzést tartalmazó csoportot lehet beállítani. A csoporthoz újabb jelzést hozzáadni a sigaddset, meglévő jelzést kitörölni pedig a sigdelset függvénnyel lehet. A jelzéscsoportra blokkolási parancsot vagy a blokkolás felengedésére vonatkozó parancsot adhatjuk ki a sigprocmask függvénnyel.
