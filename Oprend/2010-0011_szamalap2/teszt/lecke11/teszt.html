Önellenőrző feladatok

Kérjük oldja meg az alábbi feladatokat!

1. feladat — Mire használható az osztott memória?
1. Minden memória osztottnak is tekinthető, így egy program tetszőleges memória rekeszt elérhet.
2. A szemafórok tárterülete az osztott memória!
3. Tetszőleges adat beleírható, ezzel a programok információt tudnak megosztani egymással.
4. Speciális virtuális memória, az MMU használja!

2. feladat — Mire nem használható az shmctl függvény?
1. Törölheti az osztott memóriát.
2. Beállításokat végezhet az osztott memórián.
3. Definiálhat osztott memóriát.
4. Lekérdezhetjük az osztott memória adatait.

Vissza a tartalomjegyzékhez

3. feladat — Mire jó az üzenetsor?
1. A talk paranccsal tudunk üzeneteket írni egy másik felhasználónak.
2. Folyamatok tudnak egymásnak adatokat (üzeneteket) küldeni egymásnak.
3. Ezt csak az operációs rendszer használja, elektronikus üzenetek továbbítására jó.
4. Folyamatok tudnak e-mail üzeneteket küldeni egymásnak.

Vissza a tartalomjegyzékhez

4. feladat — Milyen adatokat küldhetünk üzenetsoron keresztül?
1. Tetszőleges típusú adatot.
2. Csak egész számokat használhatunk üzenetként.
3. Karaktersor lehet az üzenet.
4. Csatolt állomány lehet az üzenet.

Vissza a tartalomjegyzékhez

5. feladat — Mire használható a név nélküli csővezeték? (Pipeline)
1. Két tetszőleges folyamat adatokat tud átadni egymásnak.
2. Szülő-gyerek folyamat tud egymásnak adatot küldeni.
3. Gyerek-gyerek folyamat tud egymásnak adatot küldeni.
4. Az ütemezőnek tud egy folyamat adatot átadni!

Vissza a tartalomjegyzékhez

6. feladat — Mely állítás(ok) igaz(ak) a nevesített csővezetékre? (FIFO)
1. Két tetszőleges folyamat adatokat tud átadni egymásnak.
2. Osztott memóriához hasonló adatátadást használ a nevesített csővezeték.
3. Valójában egy fájlt jelent a nevesített csővezeték.
4. Gyerek-gyerek folyamat tud egymásnak adatot küldeni. A cső neve a szülőfolyamat.

Vissza a tartalomjegyzékhez

7. feladat — Mi a socket (foglalat)?
1. Input-Output eszközök szabványos kezelőfelülete.
2. A nyomtató sor csatlakozási pontja.
3. Két folyamat hálózati kommunikációjának alapja, a kommunikáció végpontjának foglalata.
4. Nevesített csővezeték csatlakozási pontja.
