Önellenőrző feladatok — megoldás

Az alábbiakban megtekintheti a feladatok megoldásait.

1. feladat — Mit nevezünk monoprogramozásnak?
1. Mikor a rendszerben csak egy program fut.
2. Mikor e rendszerben csak egy program és az operációs rendszer fut.
3. A programozás monolitikus modelljét, mikor nem moduláris a program felépítése.
4. A LINUX Mono csomag használatát!

Vissza a tartalomjegyzékhez

2. feladat — Mi a virtuális memória ?
1. Az a memória, ami rendelkezésre áll, csak az alkalmazás nem használja.
2. A fizikai memória egy kiterjesztése. A merevlemezen helyezkedik el.
3. A fizikai memória olyan része amit az operációs rendszer használ.
4. Nincs ilyen memória.

Vissza a tartalomjegyzékhez

3. feladat — Mi az NRU algoritmus?
1. Felhasználó kezelő algoritmus (Not Registrated User)
2. Folyamat ütemező algoritmus (Not RUn).
3. Virtuális memória kezelő algoritmus (Not Recently Used)
4. Lemez kezelő algoritmus.

Vissza a tartalomjegyzékhez

4. feladat — Mik a regiszterek?
1. A memória speciális része, az I/O eszközök használatát segítik.
2. A processzor cache memóriája.
3. A processzor memória rekeszei, a gépi kódú utasítások javarészt ezen memória elemeken működnek.
4. A HDD gyorsítótára.

Vissza a tartalomjegyzékhez

5. feladat — Mi a közvetlen memória címzés?
1. Mikor a címet közvetlenül egy regiszterrel adjuk meg.
2. Mikor egy memória értéket egy konstanssal adjuk meg.
3. Mikor egy index regiszterrel hivatkozunk a memóriára.
4. Nincs ilyen címzés.
